SVG Table Editor - Responsive Test

Test Instructions

  1. Open the main app at http://localhost:5174
  2. Add some text to cells - try short text and very long text
  3. Look for the "Responsive" checkbox in the Export section of the toolbar
  4. Try exporting with responsive OFF (fixed size)
  5. Try exporting with responsive ON (should adapt to content)
  6. Compare the SVG files or preview

Expected Behavior

  • Fixed Mode (responsive OFF): All cells should be 120px wide and 28px tall
  • Responsive Mode (responsive ON): Cell widths should vary based on text content
  • Long text should make cells wider
  • Short text should make cells narrower (but not below minimum)

Manual Test Cases

Test Case 1: Short vs Long Text

  • Cell A1: "Hi" (short)
  • Cell B1: "This is a very long text that should make the cell much wider" (long)

Expected Result:

  • Fixed mode: Both cells same width (120px)
  • Responsive mode: B column much wider than A column
